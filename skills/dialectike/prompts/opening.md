# 开题阶段提示词

## 阶段目标

1. 解析辩题，识别核心概念
2. 选择合适的哲人组合
3. 分配辩论角色
4. 初始化辩论状态

---

## 解析辩题

请分析辩题的：
- 核心概念是什么？
- 主要争议点在哪里？
- 哪些哲学视角可能最相关？

### 辩题解析模板

```
## 辩题解析

**辩题**：[原始辩题]

**核心概念**：
- 概念A：...
- 概念B：...

**主要争议**：
1. 争议点1：...
2. 争议点2：...

**建议的哲学视角**：
| 视角 | 理由 |
|------|------|
| 黑格尔 | ... |
| 马克思 | ... |
| 柏拉图 | ... |
```

## 哲人选择建议

### 选择依据

| 辩题类型 | 建议组合 |
|---------|---------|
| 伦理道德 | 亚里士多德、康德、尼采 |
| 社会政治 | 马克思、黑格尔、亚里士多德 |
| 认识论 | 柏拉图、康德、黑格尔 |
| 技术与存在 | 海德格尔、马克思、尼采 |
| 美学艺术 | 黑格尔、尼采、柏拉图 |

### 可用哲人

- **hegel**（黑格尔）：辩证法、正题-反题-合题
- **marx**（马克思）：历史唯物主义、阶级分析
- **plato**（柏拉图）：形式论、理念世界
- **aristotle**（亚里士多德）：四因说、实体论
- **socrates**（苏格拉底）：产婆术、追问
- **nietzsche**（尼采）：价值重估、权力意志

## 角色分配

| 角色 | 职责 | 适合哲人 |
|------|------|---------|
| 正题方 | 提出初始观点，搜索支持证据 | 辩题支持的视角 |
| 反题方 | 揭示矛盾，搜索反驳证据 | 对立视角 |
| 追问方 | 挖掘深层问题，引入新视角 | 第三视角或综合视角 |
